Custom Properties

It can be handy to pass custom information along to your policies (metadata, jquery objects, etc.). Any options passed along are made available via the options property on the continuation. Let's take a look at an example: 

  1. $.ajax({
  2.     url: 'test/something',
  3.     type: 'get',
  4.     options: {
  5.         el: $('#my-element')
  6.     },
  7.     continuationSuccess: function(continuation) {
  8.         continuation.options.el.html('Hello, World');
  9.     }
  10. });

It can be beneficial to apply these options conventionally. The $.continuations.continuation object provides a helper function for matching on properties:

  1. var customPropertyPolicy = {
  2.     matches: function(continuation) {
  3.         return continuation.matchOnOption('el', function(el) {
  4.             return el.size() != 0;
  5.         });
  6.     },
  7.     execute: function(continuation) {
  8.         continuation.el.html('Hello, World');
  9.     }
  10. };
  11.  
  12. $.continuations.applyPolicy(customPropertyPolicy);
